`yaxis` (component)
===================

Renders a horizontal time axis

Props:
    * align - if the axis should be draw as if it is on the left or right (defaults to left)
    * scale - a d3 scale that defines the domain and range of the axis

Props
-----

### `absolute`

defaultValue: `false`


### `align`

type: `string`
defaultValue: `"left"`


### `format`

defaultValue: `".2s"`


### `id`

defaultValue: `"yaxis"`


### `labelOffset`

defaultValue: `0`


### `max`

defaultValue: `1`


### `min`

defaultValue: `0`


### `style`

defaultValue: `{
    labelColor: "#8B7E7E", // Default label color
    labelWeight: 100,
    labelSize: 12
}`


### `transition`

defaultValue: `0`


### `type`

defaultValue: `"linear"`
